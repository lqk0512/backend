1. Yêu cầu môi trường
   
Node.js ≥ 18
npm ≥ 9
(Tuỳ chọn) jq để format JSON khi chạy script test

2. Cài đặt dependencies
npm install

3. Chạy server
Chạy API backend:
node app.js

Mặc định server chạy tại:
http://localhost:3000

4. Chạy script test API
   
chmod +x test-api.sh

./test-api.sh   

chmod +x test.sh

./test.sh
