1. Yêu cầu môi trường

Node.js ≥ 18

npm ≥ 9

(Tùy chọn) jq – dùng để format JSON khi chạy các script test

2. Cài đặt dependencies

Chạy lệnh:

npm install

3. Khởi chạy server

Chạy API backend bằng lệnh:

node app.js


Mặc định server sẽ chạy tại:

http://localhost:3000

4. Chạy các script test API
Cấp quyền thực thi:
chmod +x test-api.sh
chmod +x test.sh

Chạy script:
./test-api.sh
./test.sh
